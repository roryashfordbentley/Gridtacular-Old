Flexbones Grid, What is is?

The grid system is incredibly customisable via a Sass map of arguments. If you have ever used a javascript module that uses object based parameters you will be right at home.

Getting started

First you need to add the sass files to your project. You only need _flexbones-grid.scss and _grid-settings.scss. Both files need to be imported into your main Sass stylesheet, there is an example of how to do this in : style.scss

Grid Setup

The most basic setup looks something like this:

                    $grid_args:(
                        config: (
                            gridclass: 'grid'
                            columnclass: 'span--',
                            pushclass: 'push--'
                        ),
                        grids: (
                            mygrid: (
                                columns: 12,
                                gutter: 2rem
                            )
                        )
                    );

                    @include grid_generate($grid_args);
